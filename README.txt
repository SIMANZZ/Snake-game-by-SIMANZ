The snake game by SIMANZ

VERY IMPORTANT:
Your image must have the name 1,the file extension of image .jpg and be in the same folder with Snake.py


How to play?
- Movement of snake(use W,A,S,D):
- W - up
- S - down
- A - left
- D - right

- You need to eat the food(apple) and then your snake and score will rise up and so on until you lose

How can you lose?
- You lose if the snake goes off screen or crosses itself
